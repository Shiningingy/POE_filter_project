# Designer theme palette (received 2026-07-18). Role-keyed via `# @theme:<slug>` tags —
# machine-parseable: every SetTextColor/SetBackgroundColor/SetBorderColor/SetFontSize
# line carries its role slug. Colors are filter-style RGB[A] (space-separated).
#
# Semantics the designer encodes by OMISSION (see comments inline):
#   * no SetTextColor line       -> text inherits the item's rarity colour
#   * no SetBackgroundColor line -> the game's default label background
#   * SetBorderColor 0 0 0 0     -> no visible border (alpha 0)
# Our theme represents these as the sentinels "inherit" (TextColor) and "default"
# (BackgroundColor); both generators + styleResolver omit the line for them.
#
# INTEGRATION STATUS:
#   [x] campaign-gear-t1/t2/t3 -> sharket_theme.json "Campaign" Tiers 2-7 (Slice A, 2026-07-18)
#   [ ] 1:1 category roles (currency, shards, gems, div cards, essences, scarabs,
#       catalysts, delirium, heist, expedition, oils, fossils, resonators, map
#       fragments, jewels) — apply after the 3.29 content/tier work settles (Slice B)
#   [ ] new concepts needing generator plumbing: equipment-bases (by rarity),
